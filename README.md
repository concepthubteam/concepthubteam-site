# Concept Hub Team website (Next.js + Tailwind)

## Install
```bash
npm i
```

## Run locally
```bash
npm run dev
```

## Deploy (recommended): Vercel
- Import repo
- Add domain: concepthubteam.ro
- Update DNS records in your domain provider

## Replace placeholders
- public/favicon.ico (add your real favicon)
- src/components/navbar.tsx (replace square with your logo)
- src/app/contact/page.tsx (replace Formspree URL or use mailto)
- src/app/case-studies/page.tsx (replace with real cases + numbers)
